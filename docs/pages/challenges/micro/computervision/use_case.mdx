# Use Case

This page gives you a quick overview over the SQlab Profiler project that serves as an example for our computervision use cases.

## About:
The SQlab Profiler is an app that serves as a supportive sales tool for specialist retailers. Through a targeted survey and digital body measurement, suitable bikes, accessories and clothing can be recommended to each customer.

For more information, visit the Landing Page here: https://profiler.bike/en/
